# Phaser 3 maze demo

procedural generated maze in Phaser 3

### Demo

Play a demo here: https://quinten.github.io/phaser3-maze-demo/

## Running the code

### 1. Install dependencies

```
npm install
```

### 2. Local development

```
npm start
```

### 3. Build package

```
npm run build
```

### 4. Push pub folder to gh-pages

```
git subtree push --prefix pub/ origin gh-pages
```
